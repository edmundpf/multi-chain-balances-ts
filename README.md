# multi-chain-balances-ts
> Get Balances from Ethereum, BSC, Polygon Blockchains and Beefy.Finance Vaults.

## Install
``` bash
$ npm i -S multi-chain-balances-ts
```

## Usage
``` javascript
const wallet = new MultiChain()
await wallet.driver()
```
